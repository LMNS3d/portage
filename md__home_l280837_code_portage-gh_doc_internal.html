Installation instructions

Below we list copy & paste instructions for several machines. You can easily adapt them for other machines.

Darwin

Execute the following from the portage root directory:

1 {c++}
2 # darwin-fe
3 module load openmpi/1.10.0-intel_15.0.3 cmake
4 JALI_INSTALL_PREFIX=/path/to/inst-jali
5 mkdir build
6 cd build
7 cmake \
8  -D CMAKE_C_COMPILER=`which mpicc` \
9  -D CMAKE_CXX_COMPILER=`which mpiCC` \
10  -D CMAKE_BUILD_TYPE=Debug \
11  -D ENABLE_UNIT_TESTS=True \
12  -D ENABLE_APP_TESTS=True \
13  -D ENABLE_MPI=True \
14  -D ENABLE_MPI_CXX_BINDINGS=True \
15  -D Jali_DIR:FILEPATH=$JALI_INSTALL_PREFIX/lib \
16  ..
17 make -j16
18 ctest -j16 --output-on-failure

Varan

Execute the following from the Jali root directory:

1 {c++}
2 # barugon
3 export MODULEPATH=""
4 . /opt/local/packages/Modules/default/init/sh
5 module load intel/15.0.3 openmpi/1.6.5 cmake
6 JALI_INSTALL_PREFIX=/usr/local/codes/ngc/private/jali/0.9.2-intel-15.0.3-openmpi-1.6.5-rh6
7 mkdir build
8 cd build
9 cmake \
10  -D CMAKE_C_COMPILER=`which mpicc` \
11  -D CMAKE_CXX_COMPILER=`which mpiCC` \
12  -D CMAKE_BUILD_TYPE=Debug \
13  -D ENABLE_UNIT_TESTS=True \
14  -D ENABLE_APP_TESTS=True \
15  -D ENABLE_MPI=True \
16  -D ENABLE_MPI_CXX_BINDINGS=True \
17  -D Jali_DIR:FILEPATH=$JALI_INSTALL_PREFIX/lib \
18  ..
19 make -j2
20 ctest -j2 --output-on-failure

If you want to build an app that uses FleCSI, you can link against a built verison of FleCSI on Varan. An example is below:

1 {c++}
2 # barugon::flecsi
3 export MODULEPATH=""
4 . /opt/local/packages/Modules/default/init/sh
5 module load gcc/5.3.0 openmpi/1.6.5 cmake
6 JALI_INSTALL_PREFIX=/usr/local/codes/ngc/private/jali/0.9.2-gcc-5.3.0-openmpi-1.6.5-rh6
7 FLECSI_INSTALL_DIR=/usr/local/codes/ngc/private/flecsi-gcc
8 mkdir build-flecsi
9 cd build-flecsi
10 cmake \
11  -D CMAKE_C_COMPILER=`which mpicc` \
12  -D CMAKE_CXX_COMPILER=`which mpiCC` \
13  -D CMAKE_BUILD_TYPE=Debug \
14  -D ENABLE_UNIT_TESTS=True \
15  -D ENABLE_APP_TESTS=True \
16  -D ENABLE_MPI=True \
17  -D ENABLE_MPI_CXX_BINDINGS=True \
18  -D Jali_DIR:FILEPATH=$JALI_INSTALL_PREFIX/lib \
19  -D FLECSI_INSTALL_DIR:FILEPATH=$FLECSI_INSTALL_DIR \
20  ..
21 make -j2
22 ctest -j2 --output-on-failure
