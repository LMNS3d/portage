Constructor for running the Swarm driver where the kernel_type and support geometry are the same for every particle.

Parameters
[in]sourceSwarmA SourceSwarm to the source swarm.
[in]sourceStateA SourceState for the data that lives on the source swarm.
[in]targetSwarmA TargetSwarm to the target swarm.
[in,out]targetStateA TargetState for the data that will be mapped to the target swarm.
[in]smoothing_lengthsVector of smoothing lengths for each target particle - the three levels of vectors allow for a polygonal/polyhedral shaped support around each target particle and a vector of smoothing lengths for each facet of the polygonal/polyhedral support
[in]kernel_typeThe type of weight function kernel (B4, SQUARE, etc) for any target particle
[in]geom_typeThe geometry of the support (ELLIPTIC, TENSOR, FACETED) for any target particle

smoothing_lengths must have the size of SourceSwarm if center is Scatter and of TargetSwarm if center is Gather.
