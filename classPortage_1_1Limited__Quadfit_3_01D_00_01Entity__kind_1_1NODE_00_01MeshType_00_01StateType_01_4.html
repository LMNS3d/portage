Implementation of Limited_Quadfit functor for NODEs Limited _Quadfit - Fit to a field to a quadratic multinomial using a Least-Squared fit. Returns an array of parameters. If the MODE is on a boundary, the stencil is too small, so it drops to linear order. Uses an SVD decomposition for the LS regression.
